# 🐉 Interactive Dragon Cursor

A smooth, interactive dragon cursor built with HTML5 Canvas and JavaScript. 
Move your mouse to control the dragon — click to dash!

## 🚀 Live Demo
[View it live here](https://YOUR_USERNAME.github.io/interactive-dragon-cursor/dragon_cursor.html)

## 🛠️ How It Works
- **Inverse kinematics** — each segment follows the previous one
- **Smooth interpolation** — organic, fluid motion
- **Different body parts** — head, fins, spine, and tail
